# Avala TypeScript SDK + MCP Server

Official TypeScript SDK and MCP server for the [Avala API](https://docs.avala.ai).

## Packages

| Package | Description |
|---------|-------------|
| `@avala/sdk` | TypeScript SDK for the Avala API |
| `@avala/mcp-server` | MCP server for AI assistant integrations |

## Development

```bash
npm install
npm run build
npm test
```
